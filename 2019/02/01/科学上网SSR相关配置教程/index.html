科学上网SSR相关配置教程

发表于 2019-02-01 | 分类于 随手记 | | 热度:
字数统计: 1.5k | 阅读时长 ≈ 5

前言

VPN——虚拟专用网络，就是在公用网络上建立专用网络，进行加密通讯。它利用已加密的通道协议（Tunneling Protocol）来达到保密、发送端认证、消息准确性等私人消息安全效果。这种技术可以用不安全的网络（例如：互联网）来发送可靠、安全的消息。需要注意的是，加密消息与否是可以控制的。没有加密的虚拟专用网消息依然有被窃取的危险。

SS——SS作者是clowwindy，大约两年前，他自己为了翻墙写了shadowsocks，简称ss或者叫影梭。后来被请去喝茶了，然后自己也删除了github上的源代码。

SSR——自从SS作者被请去喝茶没多久，github上出现了一个叫breakwa11(破娃)的帐号，发布了SS的升级版——SSR，全称shadowsocks-R。SSR不仅兼容SS，而且还改进了混淆和协议，使数据更不容易被防火墙检测，翻墙也不那么容易被发现。

简单了解了SS和SSR的区别，我们来使用SSR吧！

fisrt step 购买一台海外服务器

网上罗列了一堆服务器Bandwagon、vultr、hostodo、KVM，我这里用的是KVM（稳定、便宜），相关链接在这里买买买
我选购的是最便宜的一款(人穷志短)

付款方式支持支付宝和信用卡，1年前可以使用支付宝，现在好像不行了，续费我用了visia的信用卡～
好了，说了一堆废话，继续往下看。

second step 服务器安装系统并连接vps服务器

安装服务器系统

先在网页上登录服务器，地址是你购买服务器的地址。
登录成功后打开服务器安装系统，我这里安装的是debian-7-x86_64(据说有利于一键安装SSR)

接着连接vps服务器

windows系统可以用XShell

名称：随意填写
协议：SSH
主机：你的服务器的IP（买服务器的时候会有一封邮件或在KVM网站上查看）
端口：服务器默认端口，安装系统后会出现。

ip和端口在下图可以查看到。

点击确认后，输入用户名（root）和密码（密码在KVM安装系统的时候可以查到）。

mac os 用命令行SSH连接

1、sudo su - （授权）
2、ssh -p ['poor'] root@['ip'] (注意空格)
事例：ssh -p 10086 root@23.2.3.1

third step 部署SSR

安装脚本

wget -N --no-check-certificate https://raw.githubusercontent.com/ToyoDAdoubiBackup/doubi/master/ssr.sh && chmod +x ssr.sh && bash ssr.sh

※复制上方代码粘贴到 Xshell 点击回车键执行（macOX 则是粘贴到工具“终端”），输入 1 点击回车键开始安装 SSR，再点击回车键自动检测服务器 IP，然后根据需要填写 SSR 用户名和密码，端口 6 位数以内。

※加密方式、协议和混淆可以全部点击回车键使用默认选项，也可以根据自己的需要自己选择。翻墙君动图演示分别选择的数字是 10/1/1。（推荐 10/1/1 或者 15/2/1 可以降低被墙的概率， 15（ chacha20）加密方式，需要额外安装依赖 libsodium）

※SSR 账号速度限制、流量限制、同时在线设备数量限制以及端口限制，根据自己的需要添加或全部点击回车键默认不做限制。

稍等片刻，安装成功。
接着来一波傻瓜式操作
在命令行中输入

bash ssr.sh

基本简单了解下就懂了，1、2、3、4傻瓜式操作。

启动BBR加速

BBR 是谷歌开源的加速算法，可以极大的提高 SSR 的速度，安装与否的速度最高能相差十几倍。
这一步我觉得有必要，加速后和未加速简直是一个天一个地。
同理输入

bash ssr.sh

输入数字“14”选择『其他功能』点击回车键执行，再输入数字“ 1”选择『安装 BBR』点击回车键执行命令；Linux 的内核版本输入 4.9.92；随后会弹出蓝色背景对话框询问是否卸载内核，使用键盘方向键选择“NO”点击回车键执行；最后会询问是否重启，输入“y”重启服务器。

多账号添加

这一步可做可不做，你搭建了可以给你的亲朋好友弄下，一起科学上网，独乐了不如众乐乐。
我是手动添加的，若你们有其他方式，分享给我哈。
过程如下：

1.执行命令cd /etc,进入etc文件夹
2.安装vim编辑器，如果已经安装，可以忽略
安装命令:yum -y install vim
3.用vim打开etc文件夹下的shadowsocks.json文件，执行命令：
vim shadowsocks.json

内容：

{
    "server":"0.0.0.0",
    "server_ipv6":"[::]",
    "local_address":"127.0.0.1",
    "local_port":1080,
    "port_password":{
      "25251":"pass25251",
      "25323":"pass25323"
    },
    "timeout":120,
    "method":"aes-256-cfb",
    "protocol":"origin",
    "protocol_param":"",
    "obfs":"plain",
    "obfs_param":"",
    "redirect":"",
    "dns_ipv6":false,
    "fast_open":false,
    "workers":1
}

server_port和password这两行，把这两行删掉。在password更改成
"port_password":{
      "25251":"pass25251",
      "25323":"pass25323"
},

注意最后行对象不要逗号。
25251、25323是你的端口，后面对应的是密码。
然后我们需要保存这个文件，首先按键盘左上角的Esc退出编辑模式，然后输入:wq并回车保存退出，退出即回到命令行模式下。
输入命令：

service shadowsocks restart

这时shadowsocksR重启，配置生效。

生效后提示：

ShadowsocksR start success

至此结束，快去给你的小伙伴分享快乐吧。
